## 2. AWT
Develop a system using Node.js with Express.js for the backend, MySQL for storing data, and React.js for the frontend. These technologies should work together to create a smooth and reliable web application.
- **Purpose**: The main goal of the system is to manage information, handle different operations, and allow easy interaction between users and the application.
- **Users**:
	1. End Users → Access services and perform tasks using the system.
	2. Administrators → Manage data, control operations, and monitor the overall functioning of the system.
- **Core Functions**:
	1. Efficiently store and manage data.
	2. Handle transactions with rollback/undo in case of errors.
	3. Provide real-time interactivity (forms, actions, dynamic updates).
	4. Generate meaningful reports for decision-making.
- **Expected Output / Reports**:
	1. Dynamic data displayed in forms.
	2. Functional interface such as user activity and operations.
	3. Proper forms with validations.

## 3. DBMS
Design and implement a DBMS Mini Project based on a real-world scenario. Your project should include the following tasks:
1. **Database Schema** – List all entities/tables with attributes and relationships.
2. **DDL (Data Definition Language)** – Write SQL statements to create tables with appropriate constraints:
	- Primary Key, Foreign Key
	- NOT NULL, UNIQUE
	- CHECK conditions
3. **DML (Data Manipulation Language)** – Write queries for:
	- Insert
	- Update
	- Delete
	- Select
4. **Transaction Management & Recovery** – Demonstrate:
	- Execution of a transaction
	- ROLLBACK in case of failure
	- Recovery using Undo/Redo logs or Write- Ahead Logging (WAL)
5. ***Expected Output / Reports*** – Present sample outputs/reports generated from your queries.
